# 🧠 Thalia — Когнитивная архитектура на основе GPT-2

*Thalia — это не просто языковая модель, а эксперимент по созданию когнитивной архитектуры с внутренней психикой, памятью и саморефлексией.

# 🔥 Ключевые компоненты

- 🧠 Динамическая психика** — 11 черт личности, 7 драйвов, система настроения, нейроконтроллер с PPO
- 💾 Hebb-память** — временные слоты с STDP-обучением, жизненным циклом и ассоциативными связями
- 🔗 Adaptive Memory** — центроидная память + мета-предиктор
- 🔄 Bidirectional Experience Exchange** — симбиоз трансформера и Mamba
- ⚓ Semantic Anchors** — стабилизация семантического пространства
- 🎯 Chain of Thought** — внутренние размышления перед генерацией
